PSR Lakehouse Documentation
============================

A Python client library for the PSR Lakehouse API, providing programmatic access to various datasets, including Brazilian energy sector entities (ANEEL, CCEE, ONS) and other domain-specific information.

Features
--------

* 📦 Simple and intuitive API for accessing multiple datasets
* 🔒 AWS IAM authentication support
* 🐼 Automatic conversion to pandas DataFrames
* 🔍 Schema discovery and exploration capabilities
* 📊 Support for filtering, grouping, and aggregating data
